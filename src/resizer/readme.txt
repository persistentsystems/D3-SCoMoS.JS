Resizer module
This is eskin pathway map specific resizer.

Features :
1) Apply Resizer to any pathway node on the fly.
2) Two modes namely 4-point and 8 point resize.
3) partially configurable through css.
Assumptions :
1) first and far most is this resizer will be used on the nodes generated
    by the d3scomos library, as it expects particular form of data be available
    on the node to be resized.
