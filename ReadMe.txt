#SBML specific Extensions over d3 library.
This JS library will consist mainly following new feature
    1) Compartments to contain the nodes.
    2) New shapes for various Species,reactions and other components.
    3) Various custom events for specific tasks

Building this library :
    1) run the gulp task "concat_d3scomos" will generate concatenated build in dist folder.
	2) running tests:
			go to folder tests open index.html.
			this  will take care of running tests and showing reports
	3) code coverage ( to be added).
